# v1.2 Protocol Matrix Proof

This proof captures a full dynamic run of TCP/UDP/SSL projections through the
FIFO-first lattice runtime using `lattice-netcat` servers and external clients.
SSL health is evaluated using ephemeral semantics (recent successful handshake).

## Runtime Environment

- lattice-runtime: `/home/main/github/lattice-runtime`
- lattice-netcat: `/home/main/github/lattice-netcat`
- Board: `examples/protocol-matrix/board.json`

## Commands Executed

```bash
# Ensure ports are free
lsof -ti :9101 -ti :9102 -ti :9103 2>/dev/null | xargs -r kill -9

# Start the protocol matrix board
PATH=/home/main/github/lattice-netcat:$PATH \
  ./lattice run examples/protocol-matrix \
  > docs/proofs/protocol-matrix-run.log 2>&1 & echo $! > /tmp/protocol-matrix.pid

# Send payloads
sleep 3
echo "tcp-ok" | nc -w 1 127.0.0.1 9101
echo "udp-ok" | nc -u -w 1 127.0.0.1 9102
echo "ssl-ok" | socat - OPENSSL:127.0.0.1:9103,verify=0 >/dev/null 2>&1

# Stop the runtime
kill $(cat /tmp/protocol-matrix.pid)
```
